# BankCore System

Высокопроизводительная распределенная система управления банковскими операциями и счетами.

## 🚀 Основные возможности

### Управление счетами и операциями
*   **Гибкий жизненный цикл счета**: Создание счетов с привязкой к конкретным пользователям, лимит — до x счетов на пользователя (x можно указать в options)
*   **Финансовые транзакции**: Пополнение баланса, снятие средств и атомарные переводы.
*   **Глубокий аудит**: Хранение истории операций в формате `JSONB`, что позволяет расширять метаданные операций без изменения схемы базы данных.

### Система инвойсов
*   Выставление счетов на оплату между пользователями.
*   Управление статусами: Создан, Оплачен, Отозван.
*   Фильтрация по участникам операций и статусам через gRPC.

### Безопасность и роли
*   **Role-Based Access Control (RBAC)**: Разделение прав на пользовательские и административные функции.
*   **JWT Authorization**: Интеграция с Keycloak для обеспечения безопасности запросов.

## 🏗 Архитектура

Проект спроектирован с использованием **луковой архитектуры**, что обеспечивает полную независимость доменной логики от внешних инструментов.

*   **HTTP Gateway**: RESTful шлюз на базе ASP.NET Core, выступающий фасадом для gRPC-сервисов. Обеспечивает генерацию OpenAPI/Scalar документации
*   **Service**: Основной сервис с gRPC api, реализуюет бизнес-логику и сохранение данных
*   **Interceptors**: Использование серверных интерцепторов для стандартизации обработки ошибок и профилирования времени исполнения запросов
*   **Middleware**: Обработка grpc исключений с помощью Middleware

## 🛠 Технологический стек

*   **Runtime**: .NET 10
*   **API**: gRPC (для service), REST (через HTTP Gateway)
*   **Database**: PostgreSQL + FluentMigrator
*   **Auth**: Keycloak (JWT)
*   **Orchestration**: .NET Aspire
*   **Observability**: OpenTelemetry (Tracing, Metrics, Logging)
*   **DI**: Microsoft.Extensions.DependencyInjection (MSDI) с модульной регистрацией

## 📊 Observability

В систему глубоко интегрированы следующие инструменты мониторинга:
*   **Structured Logging**: Логирование бизнес-событий в структурированном виде для удобного анализа.
*   **Custom Metrics**: Сбор метрик по созданию счетов, транзакциям и оплатам инвойсов.
*   **Aspire Dashboard**: Визуализация состояния системы, ресурсов и телеметрии в реальном времени.

## 🚦 Быстрый старт

Проект использует **.NET Aspire** для упрощения локальной разработки и запуска всех компонентов.

### Требования
*   .NET 10 SDK
*   Docker Desktop

### Запуск системы
```bash
git clone https://github.com/AlexeyMK0/bank-app-api.git
cd src/BankApp
dotnet run
```

После запуска перейдите в дашборд Aspire (ссылка появится в консоли) для управления сервисами и просмотра документации API (Swagger).
